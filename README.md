# Researcher Embeddedness Statement (RES) Tool

## About
The RES Tool is a comprehensive self-assessment questionnaire designed to help researchers evaluate and articulate their positioning in relation to their research community and context.

## Features
- **26-Question Assessment** covering 8 key dimensions of researcher embeddedness
- **Personalized Profiles** (Highly Embedded, Moderately Embedded, Minimally Embedded, Weakly Embedded)
- **Interactive Learning Page** with AI-powered chat about embeddedness and positionality
- **48 Academic References** from peer-reviewed sources and trusted publishers
- **PDF Report Generation** for your methodology section

## Dimensions Assessed
1. Country/Society Relationship
2. Community Connection
3. Language Proficiency
4. Research Experience
5. Field Presence
6. Community Collaboration
7. Positionality Awareness
8. Data & Methodological Constraints

## How to Use
1. Click on "Home" to start the assessment
2. Answer all 26 questions honestly
3. Review your personalized embeddedness profile
4. Explore the "Learn" page to understand key concepts with academic sources
5. Read the "About" page for tool information and guidance

## Files
- `res_questionnaire.html` - Main tool (all-in-one file)
- `REFERENCES_LIST.md` - Complete bibliography with 48 sources
